### Two major “go-to” guides for operator developers:

1. **Kubebuilder Book** (what you mentioned)
    
2. **Operator SDK User Guide** (from the Operator Framework)
    

Both use `controller-runtime` underneath, but Operator SDK adds scaffolding, packaging, OLM integration, scorecard testing, etc., so many operator developers read both.

Other honorable mentions operator folks often reference:

- **Kubernetes Operator Pattern** docs (official Kubernetes design docs)
    
- **Controller Runtime reference docs**
    
- **Operator Best Practices (from Red Hat / OperatorHub)**
